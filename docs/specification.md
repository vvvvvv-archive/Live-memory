# ｖｖｖｖｖｖ　LIVE MEMORY 開発仕様書

## 1. 設計原則

- 投稿者ではなく、記憶を主役にする
- 入力は最小限、検索は最大限
- 非公式のファン記憶アーカイブとして運営する
- 公式情報とファンの記憶を混同しない
- 現地・映像・後追い参加の記憶を分けつつ、同じライブ内に置く
- HTMLを量産せず、データ追加でページを増やせる構造にする
