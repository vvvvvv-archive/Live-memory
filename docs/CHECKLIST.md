# 公開前チェックリスト

新しいライブ、舞台、イベントを追加したら、公開前にこのチェックリストを確認してください。

## データ確認

- [ ] `data/lives/{groupId}/{liveId}.json` を作成した
- [ ] `data/lives/index.json` に登録した
- [ ] `liveId` が重複していない
- [ ] `performanceId` が同じライブ内で重複していない
- [ ] `goodsId` が同じライブ内で重複していない
- [ ] セットリストIDが重複していない
- [ ] 公式サイトURLが必要な場所に入っている
- [ ] JSONのカンマ抜け・引用符抜けがない
- [ ] `node scripts/validate-live-data.mjs` が成功する

## ページ表示

- [ ] ライブ一覧に表示される
- [ ] 年ごとに正しい順番で表示される
- [ ] コンサート詳細ページが表示される
- [ ] 総合ページが表示される
- [ ] 公演日程ページが表示される
- [ ] 各公演ページが表示される
- [ ] 映像・円盤ページが表示される
- [ ] 映像・円盤内の曲ページが表示される
- [ ] グッズページが表示される
- [ ] グッズ個別ページが表示される

## リンク・URL

- [ ] パンくずが正しく表示される
- [ ] 総合ページの曲リンクが `song.html` に進む
- [ ] 映像・円盤ページの曲リンクが `video-song.html` に進む
- [ ] グッズ一覧からグッズ個別ページへ進める
- [ ] 公演一覧から各公演ページへ進める
- [ ] 公式サイトリンクが開く
- [ ] 404エラーがない
- [ ] 旧URLが壊れていない
- [ ] URLに誤字がない

## コメント欄

- [ ] コメント欄が表示される
- [ ] Giscusが読み込まれる
- [ ] 総合ページのコメント欄が独立している
- [ ] 総合側の曲ページ同士でコメントが混ざらない
- [ ] 公演ごとのコメントが混ざらない
- [ ] 映像・円盤側の曲ページが総合側の曲ページと混ざらない
- [ ] グッズ全体とグッズ個別のコメントが混ざらない
- [ ] コメント投稿時の注意事項が表示される

## 検索

- [ ] ライブ名で検索できる
- [ ] 年で検索できる
- [ ] 曲名で検索できる
- [ ] 公演日で検索できる
- [ ] 会場名で検索できる
- [ ] グッズ名で検索できる
- [ ] 映像・円盤内の曲ページが検索に出る
- [ ] 検索結果にページ種別が表示される
- [ ] 検索結果に同じページが重複表示されない
- [ ] コメント本文が検索対象に入る
- [ ] 内部情報やURLが検索結果本文に出ない

## 表示確認

- [ ] PC表示で崩れていない
- [ ] スマホ表示で崩れていない
- [ ] 文字がカードからはみ出していない
- [ ] セクション間の余白が不自然でない
- [ ] 公式リンクがリンクとして分かりやすい
- [ ] フッターやサイト名が正しく表示される

## GitHub Pages確認

- [ ] commit済み
- [ ] GitHubへpush済み
- [ ] GitHub Actionsが失敗していない
- [ ] GitHub Pagesで公開URLを確認した
- [ ] 反映待ちの場合は数分待って再読み込みした
- [ ] 最終動作確認が完了した

## よくあるミス

- `data/lives/index.json` に登録し忘れて、ライブ一覧や検索に出ない
- `liveId` を変更して、既存コメント欄とつながらなくなる
- `performanceId` が重複して、公演コメントが混ざる
- `goodsId` が重複して、グッズコメントが混ざる
- 曲順やセットリストIDを公開後に変えて、曲コメント欄が別扱いになる
- 映像・円盤専用曲ページにしたいのに `video.setlist` を空にしている
- JSONの形式ミスでページが表示されない
- GitHub Pagesの反映待ちを不具合だと思ってしまう
- Giscus AppやDiscussionsの設定不備でコメント欄が表示されない
- 検索用コメントデータ `data/memories.json` の更新待ちを不具合だと思ってしまう
